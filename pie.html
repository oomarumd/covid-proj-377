Overview

Our pie chart page is a visualization tool designed to provide an intuitive representation of COVID-19 data. Using interactive pie charts, users can explore and analyze COVID-19 statistics for different countries. The page dynamically generates pie charts based on real-time data fetched from reliable sources.

Key Features

  • Country Selection: Users can select a country of interest from a dropdown menu.
  • Data Visualization: The selected country's COVID-19 statistics are visually presented in a pie chart format.
  • Comprehensive Insights: The pie chart illustrates the distribution of COVID-19 cases into categories such as deaths, recoveries, and active cases, enabling users to grasp the current situation at a glance.
  • Population-Based Sizing: The size of each pie chart segment is proportional to the country's population, providing context to the data visualization.
  • Hover Information: Hovering over each segment reveals detailed information about the corresponding category, including the number of cases and percentage.
